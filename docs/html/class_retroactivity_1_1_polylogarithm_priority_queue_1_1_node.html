class Retroactivity::PolylogarithmPriorityQueue< T >::Node< T >

Estrutura que representa um nó na checkpoint tree.

Em um nó é mantido uma fila de prioridade parcialmente retroativa correspondente ao intervalo contido por esse nó, e duas árvores binárias balanceadas persistentes, qnow e qdel, contendo os elementos inseridos e os tempos de deleção desses elementos respectivamente
